Commands:

launchpad new <template name>
	Bootstraps a project from given template in current directory.
	Prompts user interactively for variable values.

launchpad list
	Prints the list of all availabl template names.

launchpad version
	Prints installed tool version.

launchpad config
	Prints current configuration.

launchpad set-config <config file path>
	Instructs launchpad to read its config from file with given path.
	This command sets a LAUNCHPAD_CONFIG_PATH env. variable for current user.
